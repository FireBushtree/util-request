# envcloud-utils-request

## Install

```
yarn add envcloud-utils-request
```

or

```
npm i envcloud-utils-request
```
## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
